# ⚽ Prode Mundial

Una aplicación de pronósticos deportivos (*prode*) para la Copa del Mundo, desarrollada como un entorno de experimentación personal para dominar arquitecturas modernas de desarrollo web y el consumo de APIs de terceros.

---

## Características del Proyecto

* **Gestión de Pronósticos:** Permite a los usuarios cargar, guardar y modificar sus predicciones para los partidos del mundial.
* **Sincronización Automatizada:** Consumo e integración de datos en tiempo real (partidos, fixtures y resultados oficiales).
* **Arquitectura Serverless:** Implementación de servicios en la nube para la persistencia y manejo eficiente de datos sin necesidad de mantener un backend tradicional.

## Stack Tecnológico

El proyecto se construyó seleccionando herramientas modernas que garantizan un desarrollo ágil y eficiente:

* **Frontend:** [Vue.js] Framework progresivo utilizado para consolidar, expandir y llevar a la práctica los conocimientos técnicos desarrollados en el ámbito académico.
* **Base de Datos y BaaS:** [PostgreSQL] gestionado a través de [Supabase], aprovechando su ecosistema para el almacenamiento ágil de datos.
* **Fuentes de Datos:** Integración con la API externa de [football-data.org] para la obtención automatizada de estadísticas deportivas.
* **Despliegue y Hosting:** [Vercel] Plataforma optimizada para el despliegue continuo y entrega de aplicaciones web con alto rendimiento.

## Metodología de Desarrollo

Este repositorio nació con un enfoque puramente educativo, orientado a asimilar buenas prácticas en el ecosistema de Vue, Supabase y Vercel. 

El proceso de diseño de la base de datos, optimización de consultas y refactorización de código se llevó a cabo utilizando **Claude AI** como asistente de desarrollo, lo que permitió acelerar la resolución de problemas técnicos complejos y asegurar un código más limpio y mantenible.
